# Contributing to esxport

Thank you for your interest in contributing!

## Development

```bash
# Clone
git clone https://github.com/blink-zero/esxport.git
cd esxport

# Build
make build

# Test
make test

# Lint
make lint
```

## Guidelines

- Follow [Conventional Commits](https://www.conventionalcommits.org/)
- Write tests for new features (TDD preferred)
- Run `make all` before submitting a PR
- Keep PRs focused and small

## Reporting Issues

Please include:
- esxport version (`esxport version`)
- vCenter/ESXi version
- Steps to reproduce
- Expected vs actual behavior
